Sheet #1


Standard Templates - Used to be quickly copied into a WCP Daily or WCP Outage with its Safety Requirements, Isolation Tags

Notes:

Use the Add, Copy Buttons to Create a new Template Or To Copy an Existing. Use Copy, Copy To: Work Clearance Permit Area or Copy To: Outage

Clicking on any column heading will sort/reverse sort by that column. All columns can be filtered by applying selected filters.

Quick Use Steps:
  1. To use Copy, Copy To:, Edit, View and Print requires highlighted selection before clicking.
  2. Copy, Copy To: can handle multi selections to mass copy (use click, drag and hold or Cntr click or click and shift+click) to mass select
  3. Add will create a new templete and place you in Edit mode.
  4. Edit allows changes and additions to an existing Template, changes to Safety Requirements and Isolation Points.
  5. Templates are numerically numbered. Only admins can add, edit or delete templates

Sheet #4


Outage Status Change - Used to create (Add New) and Manage Outages

Notes:

Outages are stored with a unique name. Adding a Date to the name organizes the Outages over time.

Clicking on any column heading will sort/reverse sort by that column. All columns can be filtered by applying selected filters.

Quick Steps:
  1. Click Add New.
  2. Fill in Outage Name, Description, Start and End Dates.
  3. Click Save. Outage will be in Supend State.
  4. Change status by selecting on outage, select status from dropdown, Click Change Outage Status.
  5. ACTV and TEMPLA status can be used to copy WCP's to.

Sheet #5


Safety Requirements

Notes:

Every Edit screen has a Safety Requirement Button to access the entry and changes of safety requirements for a WCP.

Quick Steps:
  1. Click the Safety Requirement Button.
  2. Select the Safety Requirements needed by using Cntl Click, Hold and Sweep methods.
  3. Click Add requirements.
  4. To remove and Safety Requirements, Check its box.
  5. Click on the blue line at top of screen, ( To Delete Requirements Check Box Below and Click Here ...).

Sheet #6


Isolation Points

Notes:

Every Edit screen has a Isolation point Button to access the entry and changes of Isolation points for a WCP.

Quick Steps:
  1. Click the Isolation point Button.
  2. Select from the dropdowns, Enter a custom Isolation Point.
  3. Click ADD CUSTOM ENTERED ISOLATION POINT.
  4. Or Use the Type ahead method in the dropdown to locate and select an Isolation Point from the site isolation table.
  5. Click Edit or Delete on a row to modify or remove isolation point.

Sheet #7


Direct Tag Printing - COVLOTO GK420t Zebra Printer SETUP INSTRUCTIONS

Notes:

Pay attention to the proper width and height settings for your labels or direct tag printing.

Browser top margin must be set to 1.75 inches when using direct tag printing. See below.

Quick Steps:
  1. Load Labels or Tags along with a ribbon into printer (ribbon was pre-loaded)

    See this helpful YouTube: https://www.youtube.com/watch?v=MgriLAksmFk

  2. Plug in power to printer, turn printer on
  3. Press top feed button until the green light blinks twice Printer will auto align labels or danger tags
  4. Connect printer via USB, COM OR LPT cable to computer
  5. Use your windows machine system settings for Printers and Scanners to ADD a new Printer. You should look for Zebra GK420t. If you use the cd that came with the printer to setup the printer, it will use a different driver and may stop the direct tag printing from working.

Your settings should look like this:


Make any adjustments to width and height as described in notes below. Once complete Printer is ready and installed to your machine.


Notes: If you physically change a Zebra Printer, Repeat above steps.

Depending if you use labels or direct print Danger tags adjust width and height for labels, i.e. 4 X 3 labels were used for testing your printer and 2.8 X 5.43 setting for the direct danger tag.

In your browser when printing tags, all headers, dates etc. should be turned off and top margin set to 1.75 inches. Other margins are set to 0.0


You can also install your printer twice once for each type of output used by naming them differently and configuring them differently depending on your needs. Each user that produces tags may have to install the printer to their windows configuration.

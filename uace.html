Using Add, Copy and Edit

  1. Return To Home Page

Add, Copy and Edit Buttons

Add- Create a new WCP from stratch. A new WCP is created and placed in Edit Mode to complete updating.

Copy- After selecting on a WCP and clicking, a new WCP is created from the copied WCP including safety requirements and isolation points.

Edit- After selecting on a WCP and clicking, the WCP is placed in Edit Mode to accept changes, Safety Requirements and isolation points.

Add Button

Add

Creates a new WCP and placed in Edit mode.

Add works the same in both WCP area and Outage WCP area screens. Preprinted WCP (Standard). Template area Add is only available to site admins.

Copy Button

Copy - Step 1

Highlight WCP's to Copy to the WCP Area. Highlighting can be done 1 of 3 ways.

1- Click on a WCP to copy.

2- Use Cntr Click or Click and Shift+Click to select more than one WCP.

3- Click + Drag, release click to mass Select WCP's.

The Copy Button works the same across an Outage WCP Area and Preprinted WCP (Standard) Template Area. Template Area internal copy is reserved for Site Admins Only.

Use Copy to: to quickly copy similar WCP's into the WCP working and/or WCP Outage area to save on entry times. Copy To: wcp or outages can be accessed by a site user.

Copy - Step 2

The system will ask to confirm the Copy.

Copy - Step 3

The newely created WCP (created by the single (COPY) Button) is placed in Edit mode for additional updating.

Edit Button

Edit

Highlight WCP's to Edit the WCP. The Edit Button works the same across Outage WCP Area and Preprinted WCP (Standard) Template Area.

Template Area internal Edit is reserved for Site Admin Only.
